# Richmond Events Center
A responsive multi-page website built using HTML and CSS. 
This website shows the venue's information as well as their events schedule and contact form. 

# Pages
- Home - Introduction and Featured Events
- About - About us, Mission statement, and Our Team sections.
- Events - Events schedule and Contact form sections. 

# Features
- Symantic HTML structure
- Responsive layout
- Event schedule
- Contact form
